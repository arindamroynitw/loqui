üìã Permission Setup

On first launch, Loqui will guide you through granting three required permissions:

  • Microphone: To record your voice for transcription
  • Accessibility: To insert transcribed text into applications
  • Input Monitoring: To detect fn key presses

How to Use

  1. Look for the Loqui icon in your menu bar (top-right)
  2. Press and hold the fn key to start recording
  3. Speak your message
  4. Release the fn key to stop recording
  5. Loqui will transcribe your speech and automatically insert the text into your active application

The HUD timer shows your recording duration. Transcription happens on-device using WhisperKit, with AI cleanup via cloud APIs (Groq/OpenAI).

Configuration

Click the Loqui menu bar icon ‚Üí Settings to configure:

  • Groq API Key: For Llama 3.1 70B cleanup (free tier: 30 req/min)
  • OpenAI API Key: Fallback for GPT-4o-mini (when Groq unavailable)

Get API keys:

  • Groq: console.groq.com
  • OpenAI: platform.openai.com/api-keys

Build from Source

For developers who want to build Loqui from source:

See README for detailed build instructions.

Performance

  • Whisper Transcription: ~3 seconds
  • LLM Cleanup (Groq): ~0.3 seconds
  • LLM Cleanup (OpenAI): ~0.5 seconds
  • Total Pipeline: ~3.5 seconds (fn release ‚Üí text insertion)

Performance varies based on audio length and network latency for LLM cleanup.

Known Issues

  • App is unsigned (security warning expected on first launch)
  • No Mac App Store distribution (requires non-sandboxed permissions)
  • Clipboard is overwritten and not restored after text insertion
  • First launch downloads ~6 GB of models (Whisper: 594 MB, Qwen: ~5 GB - note: Qwen will be removed in v1.1)

Troubleshooting

App won't open

Follow the security workaround steps above (right-click ‚Üí Open).

Permissions denied

Go to System Settings ‚Üí Privacy & Security ‚Üí check Microphone, Accessibility, and Input Monitoring tabs. Ensure Loqui is enabled.

Transcription fails

Check that you've configured at least one API key (Groq or OpenAI) in Settings. Verify your API keys are valid.

tail -f ~/Library/Application\ Support/Loqui/logs/loqui.log
